# Seasonal-Allergy-Data

A full-stack web application that provides real-time pollen and allergen data to help users make descisions about their outdoor activities based on their location. This project really hit home as someone who suffers from seasonal allergies.

## Features
### Dashboard
• Real-time Balance Tracking
• Income Management
• Expense Tracking
• Transaction Logging
• Update History
### Budget Planning
• Choose from popular budgeting method templates.
• Custom Budgets for personal needs
• Visual Pie Chart
### Future Balance Calculator
• Financial projections over time frames

## Tech Stach
• HTML 5
• CSS3
• JavaScript
• Chart.js
